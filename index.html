Directory Commands

  • ls List files/directories
  • ls -a List all (including hidden)
  • pwd Show current directory
  • cd dir-name Enter a directory
  • cd.. Move up one directory
  • mkdir dir-name Create a new directory
  • rm -r dir-name Remove a directory

File Commands

  • nano/pico file-name - Open nano editor
  • cat file-name - Display file content
  • cat file1 file2 - Concatenate files

Users and Permissions

  • u = user (file owner)
  • g = group
  • o = others
  • a = all (user + group + others)

chmod Commands

Symbolic Mode: chmod u+x script.sh

Numeric Mode: chmod 777 file.txt

System & Utility Commands

  • who - List users
  • whoami - Display current user
  • uname - Display system info
  • date - Show date and time
  • clear - Clear terminal
  • sort - Sort lines

YUM & GCC Commands

  • yum install package - Install software
  • gcc --version - Check GCC version
  • gcc hello.c -o hello - Compile C program

Stream Redirection & Pipes

  • > - Redirect output (overwrite)
  • >> - Redirect output (append)
  • < - Redirect input
  • | - Pipe output to another command
  • 2> - Redirect errors

System Calls

  • getpid() - Get process ID
  • getuid() - Get user ID
  • fork() - Create a child process
  • wait() - Wait for child
  • exit() - Exit process

Shell Commands

  • test - Evaluate conditions
  • fi - End of if block
  • case - Switch-case block
  • esac - End of case block
